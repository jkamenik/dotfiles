# DotFiles

These are my dot files and allow me to transfer my configs between machines.

## Install

To install

```bash
$ git clone git@github.com:jkamenik/dotfiles.git
$ cd dotfiles
$ ./install.sh
...
```

## Sync

To sync changes made locally do the following

```bash
$ ./sync.sh
$ git diff
$ git add ...
$ git commit
```
